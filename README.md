# TechBlog
Technical Blog project using Java EE

This is a web-application to post technical blog.
Technology used:
  Java EE
    Servlet and JSP
    JDBC
  MySql
  Tomcat Server
  HTML, CSS & JavaScript
  Bootstrap
  JQuery

To clone the repositoy
  git clone https://github.com/akashptr/TechBlog.git

Requirement
  JDK 1.8 or later
  Tomcat server
  MySql
  Eclipse IDE(Optional)
